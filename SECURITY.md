# Security Policy

`snitch-fw` helps inspect local network exposure, but it should not be treated as the only source of truth for security decisions.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

If the repository has GitHub private vulnerability reporting enabled, use it. Otherwise, contact the maintainers privately and include:

- A concise description of the issue
- Steps to reproduce
- Affected versions or commits
- Any known workaround

## Supported Versions

Until the project reaches `1.0.0`, only the latest released version is supported.
